# dictionary
Map numbers to words

## Setup

* clone the application
* bundle
* run using $ rails s

## Concept

### Active Sorage
To upload file directly without using any gem.

### Reading data
Reading the file from stored place and pushing it to an array, finding combination based on mappings.
